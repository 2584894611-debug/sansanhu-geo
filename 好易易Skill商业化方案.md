## 一、技能定位与差异化

### 1.1 核心定位

**技能名称**：爆款口播文案

**一句话简介**：7步闭环生产各行业短视频口播文案，黄金3秒钩子+去AI味+7维度打分，低于7分淘汰。

**目标用户**：
| 用户画像 | 需求场景 | 付费意愿 |
|----------|----------|----------|
| 短视频创作者 | 提升内容产量和质量 | 高 |
| 自媒体运营者 | 批量生产口播内容 | 高 |
| 品牌方/企业 | 制作营销短视频 | 高 |
| MCN机构 | 培训旗下达人 | 中 |
| 个人创业者 | 打造个人IP | 中 |

### 1.2 差异化优势

#### 相比通用AI写作工具
| 维度 | 通用AI写作 | 本技能 |
|------|------------|--------|
| 行业适配 | 需大量提示词调教 | 行业通用，开箱即用 |
| 质量保障 | 无标准评估 | 7维度打分，7分以下淘汰 |
| 去AI味 | 无专业处理 | 专项检查清单逐项校验 |
| 黄金3秒 | 无强调 | 强制前3秒钩子设计 |
| 流程完整 | 单次生成 | 7步闭环，环环相扣 |

#### 相比人工代写服务
| 维度 | 人工代写 | 本技能 |
|------|----------|--------|
| 响应速度 | 24-48小时 | 5-10分钟 |
| 价格 | 200-2000元/条 | 订阅制低至几元/条 |
| 质量稳定 | 依赖个人水平 | 标准流程保障 |
| 批量生产 | 成本倍增 | 边际成本趋零 |

### 1.3 核心卖点

1. **7步闭环**：从搜集到输出，完整生产流程
2. **黄金3秒**：强制钩子设计，不让观众划走
3. **去AI味专项**：消除ChatGPT痕迹，像真人创作
4. **7维度量化打分**：客观评估，低于7分淘汰重写
5. **行业通用**：科技、美食、母婴、汽车、房产、金融、教育全覆盖

- [ ] 技能描述（200字内精炼版）
- [ ] 详细说明（1000字内完整版）
- [ ] 演示视频（可选，强烈推荐）

### 2.2 提交审核流程

```
步骤1：登录扣子开发者平台
  └── https://.coze.cn/open

步骤2：创建技能
  └── 选择「技能」→ 创建新技能
  └── 填写基础信息（名称、分类、标签）

步骤3：上传技能文件
  └── SKILL.md 为主文件
  └── references/ 目录下所有引用文件

步骤4：填写商店信息
  └── 一句话简介
  └── 详细描述
  └── 使用示例
  └── 常见问题
  └── 上传封面/图标

步骤5：提交审核
  └── 等待官方审核（1-3个工作日）
  └── 如有驳回，根据意见修改后重新提交

步骤6：上线发布
  └── 审核通过后选择「立即上线」或「定时上线」
  └── 设置定价策略
  └── 开启数据分析
```

### 2.3 审核避坑指南

| 审核项 | 常见问题 | 解决方案 |
|--------|----------|----------|
| 技能命名 | 与已有技能重名 | 加前缀或差异化词，如「爆款口播文案pro」 |
| 功能描述 | 夸大宣传 | 客观描述功能，不写「最强」「第一」 |
| 引用文件 | 路径错误 | 确保SKILL.md中引用的路径正确 |
| 示例内容 | 涉及版权 | 使用虚构案例，不引用真实品牌 |

## 三、定价建议

### 3.1 市场行情调研

| 竞品技能 | 功能 | 定价 | 模式 |
|----------|------|------|------|
| 文案生成类技能 | 单次生成 | 5-20元/次 | 按次 |
| 写作助手类技能 | 订阅制 | 29-99元/月 | 订阅 |
| 行业解决方案 | 垂直行业 | 99-299元/月 | 订阅 |
| 企业定制 | API接入 | 500+/月 | 企业版 |

### 3.2 定价策略

#### 方案A：阶梯订阅制（推荐）

| 档位 | 价格 | 权益 | 适合人群 |
|------|------|------|----------|
| 免费版 | 0元 | 每月3条文案 | 轻度用户/试用 |
| 个人版 | 29元/月 | 每日10条，每月300条上限 | 创作者 |
| 专业版 | 99元/月 | 无限使用+优先队列 | 高产创作者 |
| 团队版 | 299元/月 | 5个账号+数据分析 | MCN/团队 |

#### 方案B：按次付费制

| 档位 | 价格 | 权益 |
|------|------|------|
| 体验包 | 9.9元 | 5条文案 |
| 标准包 | 49元 | 30条文案 |
| 批量包 | 199元 | 150条文案 |

#### 方案C：混合模式（最优）

```
免费试用（3条/月）
    ↓ 转化
订阅付费（29元/月起）
    ↓ 升级
企业定制（按需报价）
```

**推荐组合**：
- 基础功能免费（3条/月）→ 拉新引流
- 个人版29元/月 → 主力变现
- 专业版99元/月 → 高价值用户
- 企业定制 → 利润增长点

### 3.3 定价心理锚点

| 锚定策略 | 应用 |
|----------|------|
| 相比人工代写 | "人工代写200元/条，我们29元/月无限用" |
| 相比时间成本 | "省下找选题、写作的时间，一顿外卖钱就够了" |
| 相比流量价值 | "一条爆款视频带来的流量，远超订阅费" |
| 相比机会成本 | "花29元试错，比错过爆款机会便宜多了" |
