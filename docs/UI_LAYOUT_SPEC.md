# 三三狐GEO 网站UI布局规范

> 供Codex开发参考，所有页面布局、组件、配色、间距均按此规范执行。

---

## 全局设计规范

### 配色
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #FF6B35 | 橙色，呼应三三狐logo |
| 辅助色 | #1A1A2E | 深蓝/深灰，背景、导航 |
| 强调色 | #00D4AA | 绿色，成功/提升/正向指标 |
| 警告色 | #FF4757 | 红色，下降/预警/负向指标 |
| 中性灰 | #6B7280 | 次要文字、边框 |
| 背景白 | #F8FAFC | 页面背景 |
| 卡片白 | #FFFFFF | 卡片背景 |

### 字体
- 标题：24px/20px/16px，font-weight: 600
- 正文：14px，font-weight: 400
- 数据数字：32px/24px，font-weight: 700，使用等宽字体
- 行高：1.6

### 间距
- 页面内边距：左右 max-width 1200px 居中
- 卡片间距：24px
- 卡片内边距：24px
- 圆角：8px（卡片），4px（按钮）

### 组件库
- Element Plus（Vue3）
- 图表：ECharts
- 图标：Element Plus Icons

## 页面1：首页/落地页


### 顶部导航栏
- 高度：64px，背景白色，底部1px边框
- 左侧：三三狐logo + "三三狐GEO" 文字
- 中间：导航链接 — 产品功能 | 定价方案 | AI引用榜（参考SheepGeo导航）
- 右侧：登录按钮（文字）+ 免费试用按钮（橙色填充）

### Hero区
- 高度：~500px，深蓝渐变背景（#1A1A2E → #16213E）
- 左侧文字区（60%宽度）：
  - 主标题（36px白色粗体）：「让品牌在AI搜索里被看见」
  - 副标题（18px灰白色）：「三三狐GEO帮你监测、诊断、优化品牌在9大AI平台的搜索可见度」

- 右侧插图区（40%宽度）：模拟AI搜索结果截图（展示品牌被AI引用的效果）

### 核心价值区（4个卡片横排）
- 标题：「为什么品牌需要GEO？」
- 4个等宽卡片，每个包含：
  - 图标（48px）
  - 小标题（16px粗体）
  - 说明文字（14px灰色）
- 内容：
  1. 🔍 AI搜索正在取代传统搜索 — 60%用户用AI搜索找品牌
  2. 📊 品牌可能正在"隐身" — 你的品牌在AI回答里排第几？
  3. 🔄 持续监控才能守住排名 — 竞品在动，你不动就是退
  4. 🚀 优化越早优势越大 — 先被AI记住的品牌，后来者很难超越

### 功能展示区（3个功能区，左右交替布局）
- 每个功能区：左侧文字 + 右侧模拟界面截图

### 底部CTA
